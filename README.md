使用データはリポジトリに挙げていないためプログラムは動かない。

プログラムの構成。

１，srcフォルダの中に、実験で使用する関数が記述されている。

２，src直下にあるファイル（model_basic_....pyなど）はプログラムを動かす際（モデル構築、値の計算など）の各種パーツが記述されている。

３，src/modelにあるファイルはデータとハイパーパラメータを入力すると異常値などの必要な結果を返す関数が記述されている。ここからsrc直下の関数などを呼び出す。

４，実験はdevelopにあるファイルで行う。developの中のフォルダが各実験内容に対応する。

５，ただ、最新の実験はフォルダごとに関数を記載したため実験のフォルダで完結する。

例えば「develop\search\search_skip\before\search_skip_complete」以下の実験は「search_skip」以下で実験が完結するようになっており「custom_Class」,「functions」以下に必要なモジュールが記載されている。
